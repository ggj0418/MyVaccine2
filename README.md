# MyVaccine2
모바일 백신 만들기 2회차

본 프로젝트는 다음의 기능을 목적으로 제작중에 있다.

1. 자신의 기기에 설치된 APK의 리스트 확인
2. 설치된 APK 중 보안적 위협이 되는 APK 검출 (Manifest의 퍼미션을 활용)
    -> 위험도를 퍼센티지로 표기
3. 설치된 스파이웨어 확인 및 삭제
